    Base 2 para base 10

  • Cada número binário deve ser multiplicado por uma potência de base 2, o último será igual a 2 elevado na 0, o penúltima 2 elevado na 1 e assim por diante.


    • Adição e subtração de binários

    Adição

  • - Os cálculos de adição e subtração com números binários são basicamente iguais aos calculos com números decimais.

  • - As adições funcionam da seguinte maneira:



    • 0+0=0


    1+0=1


    0+1=1


  • 1+1= 10 (para chegar a este resultado basta observar que o resultado em decimal seria igual a 2 e 2 em Binário é igual a 10) O "0" permanece enquanto o "1" é enviado para a próxima casa


    • 1+1+1=11


    Sutração

    0-0=0


    1-0=1


    1-1=0


    0-1=1 Nesse caso pega-se "emprestado" ficando 10-1=1


Octal

Definição do Sistema Octal

O sistema de numeração Octal ou também chamado de Base 8 possui oito dígitos, 0, 1, 2, 3, 4, 5, 6, 7.


A forma de arranjar os 8 dígitos para representar as diferentes quantidades é a mesma usada anteriormente. Quando os 8 símbolos são todos usados, 0, 1, 2, 3, 4, 5, 6, 7, os próximos devem ser arranjos do 1 com cada um dos 8 símbolos, então temos 10, 11, 12, 13, 14, 15, 16, 17, daí muda para os arranjos com 2, e assim por diante.(Trecho tirado do material da professora para ficar melhor explicado/Arquivo do Sistema de Numeração)

Hexadecimal

O sistema de numeração hexadecimal ou também chamado de Base 16 possui dezesseis dígitos, 0, 1, 3, 4, 5, 6, 7, 8, 9, A, B, C, D, E, F.


A forma de arranjar os 16 símbolos para representar as diferentes quantidades na Base 16 é a mesma usada anteriormente. Só que agora temos mais símbolos do que a Base Decimal. Quando os 16 símbolos são todos usados, os próximos devem ser arranjos do 1 com cada um dos 16 símbolos. (Trecho tirado do material da professora para ficar melhor explicado/Arquivo do Sistema de Numeração)



conversão

Base 10 para base 16

  • Primeiro deve-se dividir o número decimal sucessivamente por dezesseis e guardar o resultado.



  • Processo deve ser repetido até que o quociente seja igual a 16.



  • A conversão será obtida com a sequência dos restos junto com o último quociente, começando pela direita.


    • Base 16 para base 10

  • Seguindo a mesma regra dos outros sistemas, deve-se decompor o número e multiplica-lo por uma potência de base 16.


    • base 2 para base 16

  • Deve-se usar a tabela de equivalência, primeiramente separando o número binário em grupos de quatro dígitos, logo depois os substituindo por seus correspondentes na tabela de equivalência
